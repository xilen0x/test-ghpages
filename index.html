How to upload static sites to github pages!



In a new directory, open the terminal and:
$ npm init -y
Install gh-pages module: $ npm i gh-pages Create the .gitignore file and add the node_modules line in it.
Create a repo in github: Copy the line of your repo: example: git remote add origin git@github.com:user/name-repo.git and past. Initializing the repository: $ git init $ git add . $ git commit -m "first commit" $ git push -u origin master #(or main, depend of your repo)
and run: $ npx gh-pages -d . .
You can create a shortcut in your packege.json:
.
"scripts": { "deploy": "gh-pages -d ." }
.
So, after that you can call:
.
$ npm run deploy
